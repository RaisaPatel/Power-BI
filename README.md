# AI in Education System -- Power BI Dashboard

## Project Overview

This Power BI project analyzes the impact of Artificial Intelligence in
the Education System. The dashboard provides insights into AI adoption,
student performance, institutional benefits, challenges, and emerging
trends in modern education.

The objective of this project is to transform raw educational data into
meaningful visual insights to support data-driven decision-making.

------------------------------------------------------------------------

## Objectives

-   Analyze AI implementation across educational institutions
-   Evaluate impact on student performance
-   Identify trends in AI-based learning tools
-   Compare traditional vs AI-enhanced education models
-   Highlight key benefits and challenges of AI adoption

## Key Insights

-   AI integration improves personalized learning outcomes.
-   Institutions using AI tools show measurable performance
    improvements.
-   Adoption trends are increasing year over year.
-   Major challenges include infrastructure cost and training
    requirements.
